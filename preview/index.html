Quick Start

Generate audio samples with simple CLI commands:

# Simple chord midi-cli-rs generate --notes "C4:1:80,E4:1:80,G4:1:80" -i piano -o chord.wav # With timing offsets (arpeggio) midi-cli-rs generate --notes "C4:0.5:80@0,E4:0.5:80@0.5,G4:0.5:80@1" -o arp.wav # JSON input for complex sequences echo '{"tempo":120,"instrument":"strings","notes":[...]}' | midi-cli-rs generate --json -o out.wav # List instruments midi-cli-rs instruments

Note Format

PITCH:DURATION:VELOCITY[@OFFSET]

  • PITCH: Note name + octave (C4, F#3, Bb5)
  • DURATION: Length in beats (1.0 = quarter note)
  • VELOCITY: Volume 0-127
  • OFFSET: Start time in beats (optional, default 0)

Examples: C4:1:80 | F#3:0.5:100@2 | Bb5:2:60@0

Combining Tracks with External Tools

Generate separate instrument tracks, then combine with ffmpeg:

# Generate individual tracks midi-cli-rs generate --notes "..." -i strings -o strings.wav midi-cli-rs generate --notes "..." -i piano -o piano.wav midi-cli-rs generate --notes "..." -i bass -o bass.wav # Mix with ffmpeg ffmpeg -i strings.wav -i piano.wav -i bass.wav \ -filter_complex amix=inputs=3:duration=longest \ -o mixed.wav
